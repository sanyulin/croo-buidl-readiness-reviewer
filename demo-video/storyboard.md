# Demo Video Storyboard

Target length: 4 minutes 50 seconds. Language: English. Audio is optional because the video has large built-in captions and a matching `subtitles.srt`.

## Flow

1. Title: introduce BUIDL Readiness Reviewer as a CROO developer tooling agent.
2. Problem: hackathon builders can miss submission evidence even when code works.
3. Product: one structured `ReviewReport` before submission.
4. GitHub package: public repo, README, MIT license, logo, source, and submission copy.
5. Local demo: show `pnpm build`, `pnpm pre-submit`, and `pnpm demo`.
6. Review output: score, blockers, quick fixes, checklist, CAP notes, and demo script.
7. CROO service design: `src/croo/service.ts` is ready to wire into a Provider callback.
8. Security: `.env.example` has placeholders only; real keys stay local.
9. Honest status: template-ready, with live CAP evidence still pending.
10. Close: explain why the agent helps CROO builders submit cleaner BUIDLs.
